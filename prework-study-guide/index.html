HTML

  • The head element contains information about the webpage.
  • The body element represents the visable content shown to the user.
  • The h1-h6 commands make headers for the webpage.
  • The ul(Unordered list) and li(How to put something in the unordered list) commands, make unordered lists in the form of bullet points.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can change the color of objects by simply using the name of the color or, by using the color code for said color.
  • To make sure the CSS and the HTML are linked together you need to create a CSS rule. The CSS rule is making sure the section matches the class selector.

Git

  • git status: Checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A varibale is a named container that allows us to store data in ur code.
  • Control flow is the order in whih a computer executes code in a script.
  • JavaScript is a way to make a website more interactive. Things like buttons to press, or a timed picture popping up, all of that happens because of JavaScript.
